О проекте

Лоббирование в контексте PR — это системная работа по продвижению интересов организации через публичные коммуникации. Я выстраивала партнёрства, в которых выигрывают обе стороны: организация получает ресурсы и охват, а партнёр — контент, аудиторию или решение конкретной задачи.

Студии ААК

Личный вклад

  • Содействие анимационным студиям в получении государственного финансирования через информационное сопровождение проектов
  • Подготовка обоснований и презентационных материалов для грантовых заявок
  • Продвижение студий на международном уровне: коммуникация с зарубежными партнёрами и фестивалями

Результат

Студии-члены ААК получили информационную поддержку, которая усилила их позиции при получении государственного и зарубежного финансирования.

Подведомственные организации — Правительство Москвы

Личный вклад

  • Выстроила модель взаимодействия с подведомственными организациями по формуле win-win: обоюдный PR
  • Разрабатывала медиапроекты на основе фактуры и кейсов подведомственных организаций для каналов руководства
  • Обеспечивала информационное покрытие деятельности подведов на городских ресурсах

Результат

Подведомственные организации получили доступ к аудитории городских каналов, а руководство — качественный контент для своих медиаресурсов. Модель обоюдного PR стала устойчивой практикой.

Регионы ФНТР

Личный вклад

  • Работа с региональными отделениями Федерации: им — решение проблем освещаемости, федерации — информационные поводы и расширение целевой аудитории
  • Координация информационных потоков между центральным аппаратом и региональными структурами
  • Рост количества заявок на проведение мероприятий из регионов за счёт медийной активности

Результат

Регионы получили системное информационное покрытие, которое помогло им привлечь аудиторию и спонсоров. Центральный аппарат ФНТР — постоянный поток контента из регионов для своих медиаканалов.
